## Description

Kafka poc

## Installation

```bash
$ npm install
```

## Running the app

```bash
# start dependencies (separate terminal)
$ npm run deps:start

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
